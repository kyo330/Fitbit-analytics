Customer Segmentation and Marketing Intelligence
Dataset: Customers Dataset (Kaggle - datascientistanna/customers-dataset)

Pages:
  Overview       - summary stats and data preview
  Demographics   - who the customers are
  Segmentation   - KMeans clusters with persona labels
  Relationships  - correlations and pairwise scatter plots
  Insights       - plain-language takeaways for a marketing team

To run:
  pip install streamlit pandas plotly scikit-learn scipy statsmodels
  streamlit run customer_intelligence_app.py
